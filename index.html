Conversion Toolkit

Download - Code

The Conversion Toolkit mobile application for Android provides users with an easy-to-use "toolkit" for day to day conversions. Whether you are trying to convert Kilometers to Miles, understand the amount of Electronvolts in a Joule or see how many Indian Rupees you can get for 50 British Pounds, Conversion Toolkit can do it all!

While making this app, there were many challenges I had to overcome. The currency function proved to be the most challenging, for this to work I needed live data and the application would have to display this live data into something meaningful to the user. To achieve this I used xml parsing which would retrieve the relevant information needed based on user selection. With the information parsed into a linked list the application then extracts the exchange rate value needed and multiplies it with the users input.

While it only takes a fraction of a second to download the xml data upon request, I implemented multithreading so the application does not slow down at any point for the user.
